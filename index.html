Authentication

                        

                            
Login

                            
Authenticate a user and get a token.

                            
GET ?validate&username={username}&password={password}

                            

                                
Required Parameters:

                                
    
                                    
  • username: User's username
    • 
                                    
  • password: User's password
    • 
                                

                            

                        

                    


                    

                        
Messages

                        

                            
Get All Messages

                            
Retrieve all messages.

                            
GET ?getAll&token={token}

                            

                                
Required Parameters:

                                
    
                                    
  • token: Authentication token
    • 
                                

                            

                        


                        

                            
Add Message

                            
Create a new message.

                            
POST ?token={token}

                            

                                
Required Parameters:

                                
    
                                    
  • add: Set to 1
    • 
                                    
  • data[author]: User ID
    • 
                                    
  • data[title]: Message title
    • 
                                    
  • data[message]: Message content
    • 
                                

                                
Optional Parameters:

                                
    
                                    
  • data[image]: Image URL
    • 
                                

                            

                        


                        

                            
Update Message

                            
Update an existing message.

                            
POST ?token={token}

                            

                                
Required Parameters:

                                
    
                                    
  • update: Set to 1
    • 
                                    
  • id: Message ID
    • 
                                    
  • data[author]: User ID
    • 
                                    
  • data[title]: Message title
    • 
                                    
  • data[message]: Message content
    • 
                                

                                
Optional Parameters:

                                
    
                                    
  • data[image]: Image URL
    • 
                                

                            

                        


                        

                            
Delete Message

                            
Delete a message.

                            
POST ?token={token}

                            

                                
Required Parameters:

                                
    
                                    
  • delete: Set to 1
    • 
                                    
  • id: Message ID
